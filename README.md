# Aula Colaborativa MVP

Plataforma educativa en tiempo real donde profesores y estudiantes colaboran en una pizarra compartida.

## Estructura del Proyecto

- **client/**: Aplicación Frontend (React + Vite + TypeScript)
- **server/**: Backend API y WebSocket Server (Node.js + Express + Yjs)
- **database/**: Scripts de base de datos y migraciones (SQLite)
- **docs/**: Documentación del proyecto
- **scripts/**: Scripts de utilidad para desarrollo y mantenimiento
- **uploads/**: Almacenamiento local de archivos
